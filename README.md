
## 📄 Páginas requeridas

| Página | Contenido |
|---|---|
| `index.html` | Presentación del artista, biografía, imagen destacada |
| `discografia.html` | Discos, canciones representativas, videos musicales |
| `galeria_imagenes.html` | Galería de conciertos y momentos destacados |
| `contactos.html` | Formulario de contacto |
